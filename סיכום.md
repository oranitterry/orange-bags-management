# סיכום התקדמות פרויקט — מערכת ניהול חלוקת שקיות כתומות
## עיריית ערד — יחידה סביבתית

---

## רקע הפרויקט

הפרויקט נועד להחליף פתרון קיים (AppSheet + Google Sheets) במערכת Full Stack מקצועית ומודרנית לניהול חלוקת שקיות כתומות לאיסוף פסולת אורגנית.
המערכת מנהלת **2,916 כתובות מגורים ב-12 שכונות ערד**.

---

## מה בוצע עד כה

### 1. תשתית וסביבת עבודה
- הקמת סביבת פיתוח על SD card (VHDX → תיקיית פרויקט)
- התקנת Node.js 24, MongoDB 8.2, VS Code
- הגדרת Claude Desktop עם גישה לתיקיית הפרויקט

### 2. Backend — Node.js + Express + TypeScript + MongoDB

**מבנה הפרויקט:**
```
backend/
  src/
    models/       # סכמות MongoDB
    routes/       # API endpoints
    controllers/  # לוגיקה עסקית
    middleware/   # אימות והרשאות
```

**מודלי נתונים (MongoDB Schemas):**
- `Address` — כתובת מגורים (שכונה, רחוב, בית, דירה, PropertyNumber)
- `Round` — מחזור חלוקה (שם, שנה, סטטוס, תאריכי פתיחה/סגירה)
- `DeliveryRecord` — רשומת חלוקה (קישור בין מחזור, כתובת ומתנדב + סטטוס)
- `User` — משתמש עם 3 רמות הרשאה

**API Endpoints שנבנו:**
- `POST /api/auth/login` — כניסה עם JWT
- `GET/POST /api/rounds` — ניהול מחזורים
- `PATCH /api/rounds/:id/open` — פתיחת מחזור
- `PATCH /api/rounds/:id/close` — סגירת מחזור
- `GET /api/rounds/:id/stats` — סטטיסטיקות מחזור (כולל Aggregation Pipeline)
- `GET/PATCH /api/deliveries` — רשומות חלוקה
- `POST /api/deliveries/assign` — הקצאת כתובות למתנדבים
- `GET/POST/PATCH /api/users` — ניהול משתמשים

**אבטחה:**
- JWT Authentication (תוקף 24 שעות)
- RBAC — Role Based Access Control (superadmin / admin / user)
- bcrypt להצפנת סיסמאות
- Middleware לבדיקת הרשאות על כל endpoint

### 3. ייבוא נתונים
- סקריפט Python שממיר את קובץ ה-Excel המקורי (2,916 כתובות) ל-JSON
- סקריפט Seed שמייבא את כל הכתובות ל-MongoDB
- אימות: נוצרת רשומת DeliveryRecord לכל כתובת בכל מחזור חדש

### 4. Frontend — React + TypeScript + Recharts

**דפים שנבנו:**
- `Login` — מסך כניסה מעוצב עם Hebrew RTL
- `Dashboard` — לוח בקרה ניהולי מלא

**תכונות הדשבורד:**
- 4 כרטיסי KPI (סה"כ / חולק / ממתין / לא נמסר)
- Progress Bar עם אחוז התקדמות
- Bar Chart — חלוקה לפי שכונה (Recharts)
- Pie Chart — התפלגות סטטוסים
- ניהול מחזורים — יצירה, פתיחה, סגירה
- Context API לניהול Auth

**טכנולוגיות Frontend:**
- React 18 + TypeScript + Vite
- React Router DOM לניווט
- Axios עם interceptors לטוקן אוטומטי
- Recharts לגרפים

## מערכת הרשאות

| רמה | תפקיד | סמכויות |
|-----|--------|----------|
| superadmin | מנמ"ר / מחשוב | גישה מלאה |
| admin | יחידה סביבתית / מוקד | מחזורים, מתנדבים, דוחות |
| user | מתנדב / תברואה | דיווח לכתובות מוקצות |

- בדיקות (Jest + Supertest)
